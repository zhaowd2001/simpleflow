* sgen.exe is used to generate serializer dll(AMicroBlogAPI.XmlSerializers.dll) so as to enhance XML serialization performance.

* AsyncGen is used to generated async version of the APIs. Use by run "GenAsync.bat".  (APIs in AsyncIgnoreList.txt is ignored while generating.)

* MappingGen is used to generated API mapping report. Use by run "GenMapping.bat".

* AMicroblogAPIDoc-Sandcastle.scproj is the Sandcastle project file for generating API document.
